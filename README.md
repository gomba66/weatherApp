# Weater App
### Made with ReactJS and "Open Weather API"

### Technology Stack

- React: 18.1.0
- React DOM 18.1.0
- Node v14.17.3
- npm v6.14.13

## How to run the project

 On the root project path /weatherApp

## Branches:
```
main -> for production
dev -> for development
```


Run this:
```
npm install
npm run start
```
